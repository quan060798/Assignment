
# Assignment

## Backend Set Up
Run the following command
`cd backend`
`npm install`

Copy the `.env.example` to `.env` and adjust the PgSQL port and database name at your prefrencne

`npm run orderseed` to seed some data into database

`npm run dev`

## Frontend Set Up
`cd frontend`
`npm install`
`npm run dev`
